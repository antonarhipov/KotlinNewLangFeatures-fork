# Kotlin New Language Features

Code examples exploring the latest Kotlin language features supported in IntelliJ IDEA 2026.1.

## Covered Features

- **Nested type aliases & data-flow exhaustiveness checks for `when`** — stable in Kotlin 2.3.0
- **Name-based destructuring** — destructures by property name rather than position, so reordering properties won't break code (experimental, Kotlin 2.3.20)
- **Explicit backing fields** — reduces boilerplate for properties that are writable internally but read-only to callers (experimental, Kotlin 2.3.0)
- **Unused return value checker** — warns when a meaningful return value is silently ignored (experimental, Kotlin 2.3.0)
